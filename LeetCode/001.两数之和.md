二、两遍哈希表

```
        n = len(nums)
        
        for x in range(n):
            a = target - nums[x]         
            if a in nums:             
                y = nums.index(a)
                if x == y:
                    continue
                else:
                    return x,y
                    break
            else :
                continue

```
1.由已知确定的量求未知量，本题把加变为减，用空间换时间
2.nums.index(a) ：nums序列中值为a的下标
